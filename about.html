Steps involved in creating each piece

  1. Start with a clean piece of wood 1.5 ft by 3/4 inch.
  2. Slice length wise into 3/8 inch thick strips
  3. Cut these strips into small pieces: squares, triangles, parallelograms
  4. Use a nail file to clean each piece
  5. Stain each piece
  6. Glue each piece (You may need a thin board as a backing)
  7. Use polyurethane to shine. (may need 5 coats)
